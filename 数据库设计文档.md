## 二、数据表

### 1. College（学院表）

| 字段 | 类型 | 约束 | 说明 |
|:---|:---|:---:|:---|
| id | INT | PK | 主键 |
| name | VARCHAR(100) | NOT NULL | 学院名称 |
| code | VARCHAR(20) | UNIQUE | 学院代码 |
| description | TEXT | | 学院简介 |
| created_at | DATETIME | | 创建时间 |
| updated_at | DATETIME | | 更新时间 |

---

### 2. Major（专业表）

| 字段 | 类型 | 约束 | 说明 |
|:---|:---|:---:|:---|
| id | INT | PK | 主键 |
| name | VARCHAR(100) | NOT NULL | 专业名称 |
| code | VARCHAR(20) | UNIQUE | 专业代码 |
| college_id | INT | FK → College | 所属学院 |
| description | TEXT | | 专业简介 |
| created_at | DATETIME | | 创建时间 |
| updated_at | DATETIME | | 更新时间 |

---

### 3. Student（学生表）

| 字段 | 类型 | 约束 | 说明 |
|:---|:---|:---:|:---|
| id | INT | PK | 主键 |
| student_id | VARCHAR(20) | UNIQUE | 学号 |
| name | VARCHAR(50) | NOT NULL | 姓名 |
| gender | VARCHAR(10) | | 性别 |
| id_card | VARCHAR(18) | | 身份证号 |
| phone | VARCHAR(20) | | 联系电话 |
| major_id | INT | FK → Major | 专业 |
| class_name | VARCHAR(50) | | 班级 |
| origin_province | VARCHAR(50) | | 生源省份 |
| origin_city | VARCHAR(50) | | 生源城市 |
| high_school | VARCHAR(100) | | 毕业高中 |
| status | VARCHAR(20) | DEFAULT 'pending' | 报到状态 |
| check_in_time | DATETIME | | 报到时间 |
| dynamic_code | VARCHAR(32) | UNIQUE | 动态核验码 |
| dynamic_code_expires | DATETIME | | 核验码过期时间 |
| created_at | DATETIME | | 创建时间 |
| updated_at | DATETIME | | 更新时间 |

**状态枚举**: pending(待报到) / checked_in(已报到) / cancelled(已取消)

---

### 4. DormitoryBuilding（宿舍楼栋表）

| 字段 | 类型 | 约束 | 说明 |
|:---|:---|:---:|:---|
| id | INT | PK | 主键 |
| name | VARCHAR(100) | NOT NULL | 楼栋名称 |
| code | VARCHAR(20) | UNIQUE | 楼栋代码 |
| floor_count | INT | NOT NULL | 楼层数 |
| total_rooms | INT | NOT NULL | 总房间数 |
| available_rooms | INT | NOT NULL | 可用房间数 |
| gender_restriction | VARCHAR(10) | | 性别限制 |
| created_at | DATETIME | | 创建时间 |
| updated_at | DATETIME | | 更新时间 |

---

### 5. DormitoryRoom（宿舍房间表）

| 字段 | 类型 | 约束 | 说明 |
|:---|:---|:---:|:---|
| id | INT | PK | 主键 |
| building_id | INT | FK → DormitoryBuilding | 所属楼栋 |
| room_number | VARCHAR(20) | NOT NULL | 房间号 |
| floor | INT | NOT NULL | 楼层 |
| capacity | INT | DEFAULT 4 | 床位容量 |
| current_occupancy | INT | DEFAULT 0 | 当前入住人数 |
| room_type | VARCHAR(50) | | 房间类型 |
| price | DECIMAL(10,2) | | 住宿费 |
| is_available | BOOLEAN | DEFAULT TRUE | 是否可用 |
| created_at | DATETIME | | 创建时间 |
| updated_at | DATETIME | | 更新时间 |

---

### 6. DormitoryAssignment（宿舍分配表）

| 字段 | 类型 | 约束 | 说明 |
|:---|:---|:---:|:---|
| id | INT | PK | 主键 |
| student_id | INT | FK → Student, UNIQUE | 学生 |
| room_id | INT | FK → DormitoryRoom | 分配房间 |
| bed_number | INT | NOT NULL | 床位号 |
| assignment_date | DATETIME | | 分配时间 |
| status | VARCHAR(20) | DEFAULT 'assigned' | 状态 |
| created_at | DATETIME | | 创建时间 |
| updated_at | DATETIME | | 更新时间 |

**状态枚举**: assigned(已分配) / checked_in(已入住) / cancelled(已取消)

---

### 7. Payment（缴费记录表）

| 字段 | 类型 | 约束 | 说明 |
|:---|:---|:---:|:---|
| id | INT | PK | 主键 |
| student_id | INT | FK → Student | 学生 |
| payment_type | VARCHAR(20) | NOT NULL | 缴费类型 |
| amount | DECIMAL(10,2) | NOT NULL | 金额 |
| status | VARCHAR(20) | DEFAULT 'pending' | 缴费状态 |
| order_number | VARCHAR(50) | UNIQUE | 订单号 |
| payment_method | VARCHAR(50) | | 支付方式 |
| paid_at | DATETIME | | 支付时间 |
| transaction_id | VARCHAR(100) | | 交易流水号 |
| created_at | DATETIME | | 创建时间 |
| updated_at | DATETIME | | 更新时间 |

**缴费类型**: tuition(学费) / accommodation(住宿费) / material(教材费) / other(其他)  
**缴费状态**: pending(待支付) / paid(已支付) / failed(支付失败) / refunded(已退款)

---

### 8. CheckIn（报到记录表）

| 字段 | 类型 | 约束 | 说明 |
|:---|:---|:---:|:---|
| id | INT | PK | 主键 |
| student_id | INT | FK → Student | 学生 |
| check_in_method | VARCHAR(20) | NOT NULL | 报到方式 |
| location | VARCHAR(100) | | 报到地点 |
| operator | VARCHAR(50) | | 操作人 |
| remarks | TEXT | | 备注 |
| created_at | DATETIME | | 报到时间 |

**报到方式**: manual(人工核验) / qrcode(扫码核验)

---

### 9. KnowledgeBase（知识库表）

| 字段 | 类型 | 约束 | 说明 |
|:---|:---|:---:|:---|
| id | INT | PK | 主键 |
| question | TEXT | NOT NULL | 问题 |
| answer | TEXT | NOT NULL | 答案 |
| category | VARCHAR(20) | | 分类 |
| keywords | VARCHAR(500) | | 关键词 |
| is_active | BOOLEAN | DEFAULT TRUE | 是否启用 |
| view_count | INT | DEFAULT 0 | 查看次数 |
| helpful_count | INT | DEFAULT 0 | 有帮助次数 |
| created_at | DATETIME | | 创建时间 |
| updated_at | DATETIME | | 更新时间 |

---

### 10. FAQ（常见问题表）

| 字段 | 类型 | 约束 | 说明 |
|:---|:---|:---:|:---|
| id | INT | PK | 主键 |
| student_id | INT | FK → Student | 提问学生 |
| question | TEXT | NOT NULL | 问题 |
| answer | TEXT | | 答案 |
| is_answered | BOOLEAN | DEFAULT FALSE | 是否已回答 |
| is_published | BOOLEAN | DEFAULT FALSE | 是否公开 |
| created_at | DATETIME | | 创建时间 |
| answered_at | DATETIME | | 回答时间 |

---

### 11. Announcement（公告表）

| 字段 | 类型 | 约束 | 说明 |
|:---|:---|:---:|:---|
| id | INT | PK | 主键 |
| title | VARCHAR(200) | NOT NULL | 标题 |
| content | TEXT | NOT NULL | 内容 |
| priority | VARCHAR(20) | DEFAULT 'normal' | 优先级 |
| target_audience | VARCHAR(50) | DEFAULT 'all' | 目标受众 |
| target_college_id | INT | FK → College | 目标学院 |
| target_major_id | INT | FK → Major | 目标专业 |
| is_published | BOOLEAN | DEFAULT FALSE | 是否发布 |
| published_at | DATETIME | | 发布时间 |
| expires_at | DATETIME | | 过期时间 |
| created_at | DATETIME | | 创建时间 |
| updated_at | DATETIME | | 更新时间 |

**优先级**: high(高) / normal(普通) / low(低)  
**目标受众**: all(全体) / college(按学院) / major(按专业)

---

### 12. SystemConfig（系统配置表）

| 字段 | 类型 | 约束 | 说明 |
|:---|:---|:---:|:---|
| id | INT | PK | 主键 |
| key | VARCHAR(100) | UNIQUE | 配置键 |
| value | TEXT | NOT NULL | 配置值 |
| description | VARCHAR(200) | | 说明 |
| is_public | BOOLEAN | DEFAULT FALSE | 是否公开 |
| created_at | DATETIME | | 创建时间 |
| updated_at | DATETIME | | 更新时间 |

## 三、表关系说明

| 关系 | 说明 |
|:---|:---|
| College : Major | 1对多 |
| Major : Student | 1对多 |
| Student : CheckIn | 1对多 |
| Student : Payment | 1对多 |
| Student : DormitoryAssignment | 1对1 |
| DormitoryBuilding : DormitoryRoom | 1对多 |
| DormitoryRoom : DormitoryAssignment | 1对多 |
| Student : FAQ | 1对多 |
| College : Announcement | 1对多 |
| Major : Announcement | 1对多 |
