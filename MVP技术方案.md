# 大学生新生报到智能助手 - MVP技术方案

## 一、系统概述

**核心价值**：让新生一部手机完成报到，让校方实时掌握报到进度

**技术栈**：Django 6.0.4 + DRF + PostgreSQL 18 + Bootstrap 5

---

## 二、功能清单

### 2.1 管理端（PC Web）

| 功能 | 说明 |
|:---|:---|
| 数据大屏 | 实时报到数据监控、学院/生源地统计 |
| 扫码报到 | 人工报到登记 |
| 学生管理 | CRUD、搜索、状态管理 |
| Django Admin | 全量数据管理 |

### 2.2 学生端（移动端H5）

| 功能 | 说明 |
|:---|:---|
| 登录 | 学号+手机号认证 |
| 报到状态 | 个人中心、报到状态查看 |
| 宿舍查询 | 楼栋、房间、床位信息 |
| 缴费查询 | 缴费明细、支付状态 |
| 公告列表 | 通知公告查看 |
| 常见问题 | FAQ问答 |
