# 项目总结报告

## 一、项目概述

基于 Django + PostgreSQL + Bootstrap 的高校新生报到管理系统。

## 二、技术栈

| 技术 | 版本 |
|:---|:---:|
| Django | 6.0.4 |
| PostgreSQL | 18 |
| Bootstrap | 5 |

---

## 三、已完成功能

| 模块 | 功能 |
|:---|:---|
| 学生管理 | 录入、编辑、删除、搜索、状态管理 |
| 报到管理 | 人工报到、动态码、记录查询 |
| 数据统计 | 实时大屏、学院统计、生源地统计 |
| 学生端H5 | 登录、报到状态、宿舍查询、缴费查询 |
| Django Admin | 全量数据管理 |

---

## 四、数据库（12张表）

| 表 | 说明 |
|:---|:---|
| Student | 学生信息 |
| College | 学院 |
| Major | 专业 |
| CheckIn | 报到记录 |
