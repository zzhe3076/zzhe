# 大学生新生报到智能助手 - 项目总结报告

## 一、项目概述

本项目是一个基于 Django + Vue/HTML 的大学新生报到管理系统，为高校提供新生报到、宿舍分配、缴费管理、数据统计等功能。

## 二、技术栈

- **后端框架**: Django 6.0.4
- **数据库**: PostgreSQL 18
- **API框架**: Django REST Framework
- **前端**: HTML + Bootstrap 5 + JavaScript
- **小程序**: 微信小程序

## 三、实现的功能模块

### 1. 学生管理模块
- 学生信息录入、编辑、删除
- 学号唯一性校验
- 报到状态管理（待报到/已报到/已取消）
- 专业、班级、联系方式管理
- 生源地统计

### 2. 报到管理模块
- 人工报到登记
- 扫码报到（微信小程序）
- 动态核验码生成
- 报到记录查询
- 今日报到统计

### 3. 宿舍管理模块
- 宿舍楼栋管理
- 房间信息管理
- 床位分配
- 住宿费缴纳
- 宿舍分配查询

### 4. 缴费管理模块
- 学费缴纳
- 住宿费缴纳
- 教材费缴纳
- 订单管理
- 支付状态查询

### 5. 公告通知模块
- 公告发布
- 按学院/专业定向推送
- 优先级设置
- 定时发布与过期设置

### 6. 知识库/FAQ模块
- 常见问题管理
- 分类管理
- 关键词搜索
- 查看统计

### 7. 数据统计模块（数据大屏）
- 实时报到数据监控
- 各学院报到情况统计
- 生源地统计
- 近期报到记录

### 8. 微信小程序模块
- 学生绑定
- 个人报到状态查看
- 动态报到码生成
- 扫码报到
- 缴费查询
- 宿舍分配查询

## 四、数据库表结构

### 1. college (学院表)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | Integer | 主键 |
| name | Char(100) | 学院名称 |
| code | Char(20) | 学院代码(唯一) |
| description | Text | 学院简介 |
| created_at | DateTime | 创建时间 |
| updated_at | DateTime | 更新时间 |

### 2. major (专业表)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | Integer | 主键 |
| name | Char(100) | 专业名称 |
| code | Char(20) | 专业代码(唯一) |
| college_id | ForeignKey | 所属学院 |
| description | Text | 专业简介 |
| created_at | DateTime | 创建时间 |
| updated_at | DateTime | 更新时间 |

### 3. student (学生表)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | Integer | 主键 |
| user_id | ForeignKey | Django用户(OneToOne) |
| student_id | Char(20) | 学号(唯一) |
| name | Char(50) | 姓名 |
| gender | Char(10) | 性别 |

| phone | Char(20) | 联系电话 |
| major_id | ForeignKey | 专业 |
| class_name | Char(50) | 班级 |
| origin_province | Char(50) | 生源省份 |
| origin_city | Char(50) | 生源城市 |
| high_school | Char(100) | 毕业高中 |
| status | Char(20) | 报到状态 |
| check_in_time | DateTime | 报到时间 |
| dynamic_code | Char(32) | 动态核验码 |
| dynamic_code_expires | DateTime | 核验码过期时间 |
| created_at | DateTime | 创建时间 |
| updated_at | DateTime | 更新时间 |

### 4. dormitory_building (宿舍楼栋表)

## 五、API接口列表

| 接口路径 | 方法 | 说明 |
|---------|------|------|
| /api/colleges/ | GET/POST | 学院列表/创建 |

| /api/majors/{id}/ | GET/PUT/DELETE | 专业详情/更新/删除 |

| /api/dormitory-rooms/ | GET/POST | 宿舍房间列表/创建 |
| /api/dormitory-assignments/ | GET/POST | 宿舍分配列表/创建 |
| /api/payments/ | GET/POST | 缴费记录列表/创建 |
| /api/check-ins/ | GET/POST | 报到记录列表/创建 |

| /api/announcements/ | GET/POST | 公告列表/创建 |
