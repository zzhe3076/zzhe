# 大学生新生报到系统 - 技术学习方案

## 一、学习目标

掌握全栈开发能力，独立完成 Django + PostgreSQL + Bootstrap 项目开发。

---

| 模块 | 难度 | 学时 |

| Python 基础 | ⭐ | 20h |
| Django 框架 | ⭐⭐ | 30h |
| DRF | ⭐⭐ | 15h |
| PostgreSQL | ⭐⭐ | 15h |
| Bootstrap + JS | ⭐⭐ | 25h |
| 移动端H5 | ⭐⭐ | 15h |
