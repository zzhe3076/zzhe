### ⚠️ 需要配置微信小程序才可使用的功能

所有微信小程序相关功能需要配置以下内容才能使用：

```python
# welcome_app/wechat_views.py 第21-22行
WECHAT_APPID = 'your_appid'      # 需要替换为真实AppID
WECHAT_SECRET = 'your_secret'    # 需要替换为真实Secret
```

微信小程序功能包括：
- 微信登录
- 学生绑定
- 个人报到状态查看
- 动态报到码生成
- 扫码报到
- 缴费查询
- 宿舍分配查询
