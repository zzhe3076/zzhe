# 大学生新生报到智能助手 - 技术分析

## 一、项目概述

高校数字化迎新一站式服务平台。

## 二、核心功能

| 模块 | 功能 |
|:---|:---|
| 智能问答 | 7×24小时精准AI问答 |
| 线上预报到 | 数字化报到流程 |
| 现场核验 | 扫码/人脸识别报到 |
| 数据决策 | 实时可视化大屏 |

---

## 三、技术架构

```
用户层 → Django + DRF API → PostgreSQL
```

| 技术 | 说明 |
|:---|:---|
| 后端 | Django 6.0.4 + DRF |
| 数据库 | PostgreSQL 18 |
| 前端 | Bootstrap 5 + H5 |
